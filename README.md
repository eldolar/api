#### Develop
Install dependencies
* ```$ yarn install```

Start mongodb
* ```mongod``` (mac os)

Run server
* ```$ yarn watch```

Generate ApiDoc
* ```$ yarn docs```  
Open it on `/apidoc/`

Run Test
* ```$ yarn test```
